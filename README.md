# Claude Code Mobile

スマホからClaude Code Actionを使用するためのリポジトリです。

## 使い方

1. このリポジトリでIssueを作成
2. `@claude` とメンションしてタスクを依頼
3. Claude Code ActionがMax Plan認証で自動実行

## 設定済み

- ✅ Claude for GitHub App インストール済み
- ✅ Max Plan認証情報設定済み  
- ✅ GitHub Actionsワークフロー設定済み

## 例

```
@claude この台本をGoogle Sheetsにアップロードして

タイトル: テスト台本
STEP0: ここに内容...
```

散歩中でもコーディング依頼ができます！
